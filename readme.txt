start movie-server explain :
1. First open the CMD window and switch to the movie-Server directory
2. Enter the command Node database-Initialer.js to initialize the database data
3. Enter the command Node server.js to start the server
Use files: database-Initialer.js and Server.js

start movie-front explain :
1. First open the CMD window and switch to the movie-Front directory
2. Enter the command NPM run dev to run the vue project
Ps: If the dependency is missing, execute the command NPM install

movie-front Use files:
template page, views:{home,movieDetail,otherPerson,search,register,person} and Some common components
